# Living Memory Model

## Principle

The Living Memory Model is the canonical model of Me-Wall.

It does not merely store information. It preserves and connects meaning.

## Core Objects

- Person
- Memory
- Time
- Place
- Relationship
- Organisation
- Project
- Interest
- Achievement
- Object
- Media
- Idea
- Emotion
- Thread

## Key Relationships

A Memory may:

- belong to a Person
- belong to a Time
- occur at one or more Places
- involve one or more People
- reference one or more Objects
- contain Media
- continue another Memory
- begin or conclude a Project
- express one or more Emotions
- contribute to one or more Threads

## Meaning

Meaning is discovered through the relationships between objects.

Information enters Me-Wall. Meaning emerges from its connections.
