# Technology Stack

## Status

Proposed. Not final.

## Genesis Recommendation

- Desktop-first web application
- Frontend: React / Next.js
- Styling: Tailwind CSS or similar design system
- Backend: Supabase or equivalent
- File Storage: Supabase Storage initially, Cloudflare R2 later if needed
- Hosting: Cloudflare Pages or Cloudflare Workers
- Speech-to-text: Whisper or equivalent
- Future Mobile: React Native or Flutter

## Principle

Technology is not part of the identity of Me-Wall.

Frameworks, databases, AI models, and cloud platforms may change. The Foundation and Living Memory Model must endure.
